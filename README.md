# 多人实时绘画板

##### 已支持：基本形状绘制，自由绘制，橡皮擦，线条大小控制，颜色自定义，图层选中操作（放大、缩小、旋转，拖拽），保存图片, 实时同步(多人协作)
##### 待完成：同步需要优化数据大小

体验地址： http://mysy.vip/draw/ 

#### 前端运行

```
npm install
npm run dev
```

#### 服务端运行
```
cd ./server
npm install
npm run dev
```
